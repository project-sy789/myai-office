# MyAI Office — AI Agent Team in Your Browser

A single-file web app inspired by ToonOffice.ai. Hire AI agents, assign roles, and get real work done — all running in your browser.

## Features
- 🤖 AI Agents with roles (Developer, Content, Trader, Designer, Research)
- 💬 Real-time chat with each agent
- 🧠 Powered by Google Gemini API
- 💾 Persistent storage (localStorage)
- 🌐 Thai/English bilingual
- 🎨 Dark theme, mobile-friendly

## Setup
1. Get a free Gemini API key from https://aistudio.google.com/app/apikey
2. Open `index.html` in your browser
3. Enter your API key in Settings
4. Start hiring agents!

## Deploy
Just push to GitHub and enable GitHub Pages — no build step needed!
