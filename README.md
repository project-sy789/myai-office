1|# MyAI Office — AI Agent Team in Your Browser
2|
3|A single-file web app inspired by ToonOffice.ai. Hire AI agents, assign roles, and get real work done — all running in your browser.
4|
5|## Features
6|- 🤖 AI Agents with roles (Developer, Content, Trader, Designer, Research)
7|- 💬 Real-time chat with each agent
8|- 🧠 Powered by Google Gemini API
9|- 💾 Persistent storage (localStorage)
10|- 🌐 Thai/English bilingual
11|- 🎨 Dark theme, mobile-friendly
12|
13|## Setup
14|1. Get a free Gemini API key from https://aistudio.google.com/app/apikey
15|2. Open `index.html` in your browser
16|3. Enter your API key in Settings
17|4. Start hiring agents!
18|
19|## Deploy
20|Just push to GitHub and enable GitHub Pages — no build step needed!
21|

## Quick Start
1. Open the site above
2. Click ⚙️ Settings → paste your Gemini API key (free from https://aistudio.google.com/app/apikey)
3. Click "+ Hire Agent" → pick a role → hire
4. Click the agent card → chat with your AI team!

## Roles Available
- 💻 Developer — writes code, debugs, builds apps
- 🎬 Content Creator — creates videos, posts, reels
- 📈 Trader — analyzes markets, tracks PnL
- 🎨 Designer — graphics, logos, thumbnails
- 🔍 Researcher — summarizes, analyzes, reports
- 📋 Admin — replies to customers, schedules
